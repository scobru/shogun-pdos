# Shogun NoBackend

A suite of minimalist, serverless web tools that live entirely in your browser. zero backend, zero tracking, 100% local.

## How to use

1. Open [textarea.my](https://textarea.my)
2. Type stuff
3. Marvel at the URL getting longer
4. Try to share it
5. ...
6. Profit

## Pro tips

- Start your document with `# Title` to set a custom page title
- Your data lives in localStorage AND the URL. Double the fun!
- Feeling fancy? Add a `style` attribute to the `<article>` tag via DevTools. It'll be saved in the URL too!

## Included Tools

This repository contains a suite of minimalist, local-first web tools:

- **[Textarea](note.html)**: The main tool. A minimal text editor with URL storage.
- **[Event](event.html)**: Event countdown/tracker.
- **[Habit](habit.html)**: Habit tracker.
- **[List](list.html)**: Simple checklist.
- **[Meet](meet.html)**: Meeting notes/planner.
- **[Message](msg.html)**: Distraction-free message drafter.
- **[Picture](pic.html)**: Image viewer/helper.
- **[Poll](poll.html)**: Quick poll creator.
- **[QR Code](qr.html)**: QR code generator.
- **[Secret](secret.html)**: Encrypted note sharing.
- **[Sleep](sleep.html)**: Sleep cycle calculator.
- **[Snippet](snippet.html)**: Code snippet manager.
- **[Split](split.html)**: Bill splitter.
- **[Timer](timer.html)**: Minimalist timer.
- **[Wheel](wheel.html)**: Decision wheel.
- **[Where](where.html)**: Location/coordinate helper.

*All tools operate entirely in the browser with no backend dependencies.*
